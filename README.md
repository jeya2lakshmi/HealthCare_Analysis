# HealthCare_Analysis
Healthcare Patient Data Analysis using Python &amp; Excel. Analyzed healthcare patient records using Python (pandas) and Excel to clean data, perform exploratory analysis, and generate insights such as disease trends, age-wise distribution, and gender-based patterns. Created summary reports and visualizations for decision-making.

How to Create the CSV :
- Open Notepad
- Paste the content below
- Save as patients.csv
Encoding: UTF-8
- Place it in the same folder as healthcare_analysis.py
